### electron start with react typescript
#### 这应该是最简单的electron demo了

- [x] Web唤醒
- [x] 统一菜单、事件绑定
- [x] 窗口事件
- [x] 托盘图片
- [x] 环境区分
- [x] MAC&Windows
- [x] 自动更新
- [] 单测
- [] e2e

启动web: `web:*`
打包web，输出目录为 dist `build:web:*`
打包electron js，输出目录为 app: `main:* `
打开electron开发模式: `client:*`
打包应用，输出目录为 output: `build:client:`
